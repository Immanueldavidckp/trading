2.6  YAHOO PRICE POLLER (built and running)
----------------------------------------------------------------
- New file: backend/yahoo_poller.py
- SQLite table `price_changes` with columns:
      received_at (ms-precision ISO), bar_time, tsym, yahoo_sym,
      lp, bid, ask, bid_size, ask_size,
      day_open, day_high, day_low, prev_close,
      change, change_pct, volume, source
  (bid/ask columns added later via idempotent ALTER TABLE)

- Class YahooQuoteClient
      * Manages cookie + crumb auth (auto-refresh on 401)
      * Calls v7/finance/quote in BATCH for all watched symbols
      * Returns lp + bid + ask + sizes per symbol
      * Fix applied: crumb endpoint returns 406 if Accept=application/json,
        so we send Accept: text/plain for that one call.

- Class YahooMultiPoller
      * Persists watchlist to backend/local_data/watchlist.json
      * Polls all symbols every 1s (configurable)
      * Records a row ONLY when lp OR bid OR ask changes
      * Tracks per-symbol last_price/last_bid/last_ask/change_count
      * Auto-starts on backend boot

- Static helpers
      YahooPoller.query(tsym, limit)
      YahooPoller.stats()
      resolve_symbol(query) — Yahoo search API, name → ticker mapping

- Endpoints
      POST /api/yahoo/start
      POST /api/yahoo/stop
      GET  /api/yahoo/status
      POST /api/yahoo/backfill?tsym=…&rng=1d|5d|1mo
      GET  /api/yahoo/prices?tsym=…&limit=…
      GET  /api/yahoo/stats
      GET  /api/yahoo/symbols                     — list watchlist
      POST /api/yahoo/symbols/add  {query:"…"}    — auto-resolves
      POST /api/yahoo/symbols/remove {tsym:"…"}

6. HOW TO RUN
================================================================
6.1  Backend
        cd C:\home\applicatipn\test_trad\backend
        python -m uvicorn main:app --host 127.0.0.1 --port 8000

6.2  Open in browser
        http://127.0.0.1:8000/live.html      ← live prices + bid/ask
        http://127.0.0.1:8000/                ← original dashboard

6.3  Add a new stock from the page
        Type the name OR symbol (e.g. "tcs", "hdfc bank", "tata motors")
        Press Enter / click + Add. Yahoo search resolves it automatically.

6.4  Query the SQLite directly
        sqlite3 backend\local_data\ticks.db
        SELECT received_at, tsym, lp, bid, ask FROM price_changes
        WHERE tsym='OLAELEC' ORDER BY id DESC LIMIT 20;

6.5  Switch to LIVE Shoonya (when their server is back)
        - Set MOCK_MODE=false in backend/.env
        - Restart backend
        - POST /api/recorder/start  → captures true tick-by-tick
          + 5-level depth into the `ticks` table
